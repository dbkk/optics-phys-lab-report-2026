# OpticsPhysLab Report 2026

This repository contains only the built static files for the 2026 OpticsPhysLab report assignment.

Enter the assigned eight-digit student ID in the app and report the five reproducible physical inconsistencies you observe. The app runs entirely in the browser. It does not request a name and does not send the student ID or observations to a server.

Development source, marking tools, and course records are intentionally not distributed in this public artifact repository.
